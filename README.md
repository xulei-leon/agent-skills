# agent-skills

A collection of project starter templates for different domains, each following the **OpenCode Command + Skills pattern**.

## Sub-projects

| Project | Type | Description |
|---------|------|-------------|
| `npd-sw/` | NPD-SW (New Product Development - Software) | 6-phase lifecycle template: concept → requirement → design → develop → test → launch |

More sub-projects will be added for other domains.

## Usage

Each sub-project is a standalone template. To bootstrap a new project:

```bash
./<sub-project>/install.sh init /path/to/target-dir
```

See each sub-project's `README.md` and `AGENTS.md` for details.
